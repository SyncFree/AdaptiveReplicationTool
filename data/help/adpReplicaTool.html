Adaptive Replication Tool

This tool shows graphycaly the workings of the implemented Adaptive Replication algorithms.

This tool also allows to run scripts, represented in XML format, which execution can be controlled by starting, stepping, pausing and stopping them. The description of the system composed of data centres is also contained in the test script together with the operations to execute, i.e. read and write operations.

This tool was created as part of the EU SyncFree project.
